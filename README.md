Simple ecommerce Usecase using MERN tech, used ReactJS for frontend along with Redux Saga for APi management, 
used Nodejs for backend application along with ExpressJs framework, 
used MongoDB database for storing the product and user data

How to install,
1)cd frontend
2)npm i
3)npm start

How to install backend
1)cd backend
2)npm i
3)node server.js or nodemon server.js
